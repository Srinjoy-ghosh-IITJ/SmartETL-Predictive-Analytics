\# Smart ETL Examples

This folder contains practical examples demonstrating how to use the
Smart ETL system.

\## Files Overview

\### 1. \`basic_usage.py\` - \*\*Purpose\*\*: Simple introduction to
Smart ETL - \*\*Shows\*\*: Basic data profiling, cleaning, and feature
engineering - \*\*Best for\*\*: Beginners and quick starts

\### 2. \`advanced_usage.py\`  - \*\*Purpose\*\*: Advanced features and
customization - \*\*Shows\*\*: Configuration management, pipeline
generation, feature importance - \*\*Best for\*\*: Power users and
production scenarios

\### 3. \`sales_prediction_demo.py\` - \*\*Purpose\*\*: Complete
business use case - \*\*Shows\*\*: End-to-end workflow from raw data to
model-ready features - \*\*Best for\*\*: Real-world applications and
demonstrations

\## Quick Start

Run any example: \`\`\`bash \# Basic example python basic_usage.py

\# Advanced example python advanced_usage.py

\# Sales prediction demo python sales_prediction_demo.py
